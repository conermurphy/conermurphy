# Hey, I'm Coner 👋

<img src="./header.png" title="Header Banner" alt="Header Banner"/>

I'm a **Senior Software Engineer** and **solo founder** based in Norwich, UK.

By day I'm a Senior Software Engineer. By night (and weekends) I'm building [**Doify**](https://doify.io), a tool that extends Todoist with advanced automation and organisation features. It's grown to nearly 1,000 users organically.

I'm also an [AWS Community Builder](https://aws.amazon.com/developer/community/community-builders/), I share about the things I build such as serverless architecture, TypeScript-first development, and the realities of building as a solo founder.

---

## 🚀 What I'm Building

### [Doify](https://doify.io)
Advanced automation and organisation tools for Todoist. Make your task management actually work the way it should.
